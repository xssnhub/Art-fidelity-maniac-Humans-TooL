# Art Fidelity Maniac — Humans Tool

專為藝術家提煉事物本質而存在。

這是一個以人類觀看、感知與判斷為核心的美術專用工具箱。工具保持輕量、直接、可在本機操作，不以自動判定取代人的眼睛。

## Tools

### QuickSharp Baby｜快晰寶貝

快速批量觀看與比較圖片清晰度的本機工具。

支援：

- RAW
- TIFF / TIF
- JPG / JPEG
- 鍵盤快速切換圖片
- 固定相同座標比較連續影像
- JPG、RAW、TIF 視圖切換
- 將不需要的照片移至系統垃圾桶

目前穩定版本：v1.6
